# py-card-game
Very simple blackjack written in Python.

## How to start game
Please run the following command.
```
python main.py
```

## Rules
- A can be counted as 1 only. cannot be considered as 11.
- J, Q and K is counted as 10.
- No Double down, no Split, No surrender.
